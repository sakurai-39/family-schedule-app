# Plan 7 Manual Test Checklist

Created: 2026-05-08

このチェックリストは Plan 8 のリリース準備に入る前の手動確認用。

## Preconditions

- Android development build がインストール済み。
- `npx expo start --dev-client --clear` で接続できる。
- Firebase Auth / Firestore は開発用プロジェクトを使用する。
- テストに使う Google アカウントは Ryouさん用と妻役用の2つ。

## Authentication and Pairing

- [ ] Ryouさん用アカウントで Google ログインできる。
- [ ] 初回ログイン後に呼び名を登録できる。
- [ ] 家族を作成できる。
- [ ] 招待コードを発行できる。
- [ ] サインアウトできる。
- [ ] 妻役アカウントで Google ログインできる。
- [ ] 招待コードで同じ家族に参加できる。
- [ ] 3人目の参加ができないことを確認する。

## Calendar

- [ ] カレンダー画面が表示される。
- [ ] 今日の予定タブが表示される。
- [ ] やることタブが表示される。
- [ ] 予定カードをタップして編集画面に移動できる。
- [ ] 予定を完了/未完了に切り替えられる。

## とりあえずメモ

- [ ] カレンダー画面から「メモ」画面へ移動できる。
- [ ] メモを追加できる。
- [ ] メモを「予定」に整理できる。
- [ ] メモを「期限あり」に整理できる。
- [ ] メモを「やること」に整理できる。
- [ ] 整理後のカードがカレンダー画面に表示される。

## Edit and Delete

- [ ] 予定タイトルを編集できる。
- [ ] メモ欄を編集できる。
- [ ] 担当者を変更できる。
- [ ] 予定を削除できる。
- [ ] 削除後、一覧から消える。

## Notifications

- [ ] 新しい Android development build で通知許可ダイアログが出る。
- [ ] 「許可する」を選んでもアプリが落ちない。
- [ ] 日付なしの未完了「やること」があると、カレンダー画面に `やることリストがN件あります` が表示される。
- [ ] 該当する「やること」を完了すると、バナーが消える。
- [ ] 予定・期限ありタスクを作成してもエラーが出ない。

## Security Smoke Test

- [ ] サインアウト後、ログイン画面に戻る。
- [ ] 別アカウントでログインしたとき、前のアカウントへ即ログインされずアカウント選択できる。
- [ ] Firebase Console で `households.members` が2人以内である。
- [ ] Firebase Console で `calendar_items.createdBy` が整理後も残っている。

## Known Follow-up

- 日付・時刻入力を手入力からタップ式 picker に改善する。
- 実際の通知到達は時刻をまたぐ必要があるため、必要なら短時間テスト用の通知確認モードを後続で検討する。
